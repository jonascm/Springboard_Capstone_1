Hotel Recommendation System

Welcome!
My name is Jonas Cuadrado, I'm a data scientist with a strong scientific background in math and engineering. You can check out my LinkedIn to know more about me!
This is a simple project created in the framework of the Springboard Data Science Bootcamp (you should check them out). Using python I created a recommender system for hotels, and set up this little website on AWS as a proof of concept based on this amazing tutorial. It works pretty well, and if it catches up I would like to extend the database and scale it.

Here's how it works:
• Log in or sign up. I won't send you any email ever, I promise.
• Rate a bunch of hotels you have been to. Ideally, the more the better, but with a few it still works.
• Use the search engine to get recommendations.
• Go to your favorite hotel booking website and book! (Links to come in the future - sorry!)

Let me know if you have any questions. My contact is on LinkedIn.
